What is AlienBot?

AlienBot is a Discord bot developed in Discord.js by the UFO Studios team. It includes all features for free, so nothing is locked behind a paywall! AlienBot contains a mix of useful moderation commands, fun and other misc commands to help improve your Discord server!

Commands

All command use Discord slash commands, making it much easier for you to be able to see other parameters that the commands may be able to use :D


Command Description Type
/kick Kick a member from the server. Moderation
/ban Ban a member from the server. Moderation
/unban Unban a banned member. Moderation
/warn Warn a member in the server. Moderation
/warns Check the warns of a member. Moderation
/prune Clear up to 99 messages from a channel Moderation
/about About AlienBot. General
/user-info Get info about a user. General
/server-info Get info about a server. General
/pfp Get a members profile picture. General
/bread Bread 👍 or 👎 General
namemc Get info about a Java minecraft player. General
/ping Get AlienBot's ping (and pong). General
/random-text-generator Get some random text generated for you. General
/say The bot will anonymously repeat what you tell it to say. Warning - Don't use this for bad stuff. General
/woof AlienBot will become Doggo General
/update-channel Channel where AlienBot updates are sent. Logs
/set-logging-channel Set a logging channel where message logs (and other logs too in the future) will be sent. Logs
