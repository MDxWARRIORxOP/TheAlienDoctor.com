Description

This behaviour pack tracks some basic statistics for you and outputs them to a scoreboard.

Players that have not moved for 5 minutes will be considered AFK, and will not have their stats tracked.

See below for instructions on how to use this pack! Also, if you happen to be a BigBrain person, and know how to make Bedrock UI stuff then maybe you can help make a UI for this pack? :D

How to use

  • To display
  • Displaying stats
    • To display a statistic on the sidebar, you can use the following command: "/function [StatisticName-sidebar]"
    • To display a statistic beneath a player name, you can use the following command: "/function [StatisticName-belowName]"
    • To display a statistic on the player list (the list of players on the pause screen), you can use the following command: "/function [StatisticName-list]"
    • Please note: A player will need to have operator permissions in order to run function commands.
