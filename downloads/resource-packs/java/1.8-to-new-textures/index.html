1.17 textures for 1.8

Pack description

This resource pack aims to port the modern Minecraft textures (at this time, 1.17.x) to Minecraft 1.6.1 to 1.8. The idea of this pack is to give 1.8 PvP players and 1.7.10 modded players have the modern Minecraft textures.
