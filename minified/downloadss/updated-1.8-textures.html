Description

This resource pack takes as much as possbile from the Vanilla 1.18 texture pack, and backports them to 1.8, including almost every texture! This pack is designed for 1.8 as thats what lots of servers use for PvP, however it also works in any version between 1.6.1 to 1.8.9, which means it can be used in 1.7.10, a common version used for modding.
